## **Project_Simulation**

Python-симуляции для экспериментов со статистикой, теорией вероятностей и моделированием.

Это коллекция компактных симуляций, каждая из которых демонстрирует работу того или иного математического или стохастического процесса.


### **Содержимое проекта**

```
Project_12_Simulation/
├── 01_monte_carlo/           # Метод Монте-Карло, AlphaGo, Покер
├── 02_battle_dimensions/     # Баланс для игры «Битва Измерений»
├── 03_electoral_system/      #
```

### **Технологии**

* Python 3.10
* NumPy / SciPy
* Matplotlib
